## Instructions

### Goal
Create a complete MySQL database setup script and test it.

### Requirements (Your script must include)
- **Three tables**, with **at least one foreign key relationship** between them.
- SQL statements for:
  - `DROP TABLE` (to allow re-running the script)
  - `CREATE TABLE`
  - `INSERT` (to populate the tables with sample data)
- The entire script must be **wrapped in a transaction**.

### Test Case

**Input:**  
Run your script in **MySQL Workbench**.

**Expected Output:**  
- The script runs **without errors**.
- The database contains the **three tables** you created.
- The tables include the **sample data** inserted by your script.
- Foreign key relationships are created successfully.
